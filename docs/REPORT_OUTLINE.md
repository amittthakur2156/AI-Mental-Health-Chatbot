# Final Year Project Report Outline

## Abstract
CalmMind AI is a multimodal mental wellness support chatbot that provides text chat, browser voice interaction, phone-call demo, mood analytics, journaling, safety planning and crisis detection.

## Problem Statement
Students and young adults often face stress, anxiety, loneliness and academic pressure. Existing support is not always available instantly. The project provides a safe AI-based companion for first-level emotional support and self-reflection.

## Objectives
- Build an AI chatbot for supportive conversation
- Detect mood, emotion, intent and risk
- Provide voice-based interaction
- Track mood and journals
- Provide safety protocol for high-risk messages
- Build admin analytics for research/demo

## Modules
1. User Interface
2. AI Chat Engine
3. Mood Analysis
4. Safety Detection
5. Voice Interaction
6. Phone Call Assistant
7. Journal System
8. Dashboard Analytics
9. Admin Panel
10. Database Layer

## Limitations
- Not a replacement for licensed therapy or emergency services
- Browser speech recognition support varies by browser
- Real-time phone streaming requires Twilio, public HTTPS URL and streaming STT/TTS

## Future Scope
- Native mobile app
- Multilingual professional voice models
- Human counsellor handoff
- Better AI risk classifier
- Production PostgreSQL and role-based admin auth
